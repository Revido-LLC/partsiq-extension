# How to Install the Parts IQ Extension on Chrome

Follow the steps below to install the extension for testing.

---

## Step 1 — Open the Extensions Manager

Open Google Chrome and go to the following address in the navigation bar:

```
chrome://extensions
```

---

## Step 2 — Enable Developer Mode

In the top right corner of the page, toggle on **"Developer mode"**.

---

## Step 3 — Drag and drop the extension file

Locate the `.crx` file that was sent to you and **drag it** directly into the Chrome extensions page (`chrome://extensions`).

Drop it anywhere on the page.

---

## Step 4 — Accept the installation

A confirmation dialog will appear. Click **"Add extension"** to complete the installation.

---

## Step 5 — Pin the extension to the Chrome toolbar

Click the **puzzle piece icon** (🧩) in the top right corner of Chrome.

Find **Parts IQ** and click the **pin icon** to keep it visible in the toolbar.

---

## Step 6 — Open the extension

Click the **Parts IQ icon** in the Chrome toolbar to open the side panel and start using it.

---

## Notes

- This is a test version — manual installation is required as the app is not yet published on the Chrome Web Store.
- If Chrome blocks the installation, make sure **Developer Mode** is enabled (Step 2) before dragging the file.
